# User Manual

## Learner

1. Register or sign in.
2. Complete onboarding to set the learning path.
3. Use Home to continue the next lesson.
4. Use Discover for vocabulary, grammar, image learning, and flashcards.
5. Use Practice for speaking and conversation training.
6. Use Profile to review XP, coins, streaks, achievements, and settings.

## Admin

1. Sign in with an admin account.
2. Use admin APIs to manage users and lessons.
3. Review analytics dashboard, reports, and audit logs.
4. Monitor platform health in Grafana and logs in Kibana.
